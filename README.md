# Lacia

Paclet template.

## How to use

### Install

Install from this repository:

1. download the built paclet `build/*.paclet`;

2. install the paclet:

    ``` 
    PacletInstall@File["the/path/of/paclet"];
    ```

Load the package(s):

``` 
Needs["Lacia`Init`"];
```

### Upgrade

```
PacletInstall["Lacia"];
```

### Uninstall

```
PacletUninstall["Lacia"];
```
